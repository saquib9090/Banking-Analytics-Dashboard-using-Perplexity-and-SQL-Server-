# Banking-Analytics-Dashboard-using-Perplexity-and-SQL-Server-

This project was developed with the assistance of Perplexity AI for business analysis, KPI ideation, dashboard planning, reporting concepts, and documentation support. SQL Server, Power BI, data modeling, and DAX measure implementation were performed as part of the project development process.

**AI-Assisted Analytics**
Used Perplexity AI to generate business reporting ideas.
Assisted in KPI selection and dashboard planning.
Generated analytical insights and reporting concepts.
Enhanced dashboard storytelling and business interpretation.

**DAX Measures**

## Key Metrics

- Account Count by Type
- Count of Transactions
- Customer Count by Gender
- Inactive Accounts
- Monthly Transaction Amount
- Monthly Transaction Balance
- Total Balance


**🛠️ Technologies Used**

## Technologies Used

- SQL Server
- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Perplexity AI

Conclusion

This project demonstrates how SQL Server, Power BI, DAX, and AI-assisted research tools can be combined to transform raw banking data into actionable business insights through interactive dashboards and data-driven reporting.
